# A CRUD API WITH NODE JS

    A crud api for manage pokemons. 

# FEATURES

 - Add a pokemon
 - Delete pokemon
 - View a list of all pokemon
 - update a pokemon
 - get a pokemon


# TECHNOLOGIES

 - Javascript (js)
 - Node js
 - Mysql

# REQUIRE

 - npm 
 - node
# HOW TO USE
    Execute these commands bellow:

    1. First

        git clone https://github.com/ibraum/crud_api_with_jdbc.git

    2. Second
        - npm install
